# Engineering Test Brief

Status: approved for pilot planning

Purpose
-------
This document defines the first engineering test brief for the pilot container set.

It closes `P1.6`.

## Pilot SKUs covered

- `Storage-M`
- `Dose-M`

## Target material stack

- borosilicate 3.3 glass body
- food-grade stainless steel lid structure
- platinum-cured silicone gasket

## Target ranges

### Storage-M

- nominal volume: `1.5 L`
- target wall thickness: `3.0-4.0 mm`
- target empty weight: `700-1000 g`
- allowable dimensional tolerance: `+/- 1.5 mm` on major external dimensions

### Dose-M

- nominal volume: `100 ml`
- target wall thickness: `2.5-3.5 mm`
- target empty weight: `120-220 g`
- allowable dimensional tolerance: `+/- 1.0 mm` on major external dimensions

## Pass/fail tests

### Wash-cycle survival

- repeated wash exposure without structural failure
- no unacceptable clouding, seal failure, or thread deformation

### Low-height drop

- survive low-height handling drops representative of depot and household accidents
- no crack, chip, or closure disengagement under pass threshold

### Leak resistance

- no meaningful leakage in normal upright and short-duration transport orientation tests

### Stacking stability

- stable stacking under expected fridge and route-loading conditions
- no rocking or unsafe top-surface instability

## Inspection requirements after testing

- inspect rims for chips
- inspect base for micro-cracking
- inspect threads and lid fit
- inspect gasket compression behavior and reseal performance

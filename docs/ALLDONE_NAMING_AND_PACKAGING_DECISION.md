# Glass Packaging + AllDone Naming and Concept

Source basis: condensed from the chat export originally captured for this topic and normalized to the final decision.

## Final decision

The system name is **AllDone**.

`PrecisionDelivery` is retired as the primary name and concept. It overemphasizes routing and delivery mechanics. The actual value proposition is broader: the system removes the grocery, packaging, disposal, and return burden from the user.

## What AllDone is

**AllDone** is a closed-loop home provisioning system.

It is not just "delivery." The system includes:

1. food production or sourcing
2. packaging into reusable containers
3. transport to the home
4. use by the customer
5. return, washing, refill, and recirculation

The real operating model is:

**production -> packaging -> circulation -> home supply -> return -> reuse**

## Default planned packaging feature

**Reusable glass packaging is a planned default feature of AllDone.**

This is the default packaging direction for the system because glass supports the model well:

* glass is chemically inert for normal food use
* glass does not leach additives into contents
* glass does not retain odors, oils, or staining like plastic often does
* glass is highly reusable when collection and handling are built into the loop
* glass is fully recyclable and fits a zero-waste or near-zero-waste container system
* thick reusable glass containers support a durable, premium, returnable packaging standard

Operationally, the plan is:

**AllDone uses reusable glass containers by default, with collection, washing, and recirculation designed into the system.**

## Why the name changed

`AllDone` matches the user-facing outcome better than `PrecisionDelivery`.

The promise is not:

* precise logistics
* optimized routing
* better delivery technology

The promise is:

* no shopping trip
* no carrying
* no disposable packaging burden
* no need to sort, throw away, or manage excess packaging
* no need to manage the loop manually

The user experience is:

**take -> use -> give back -> AllDone**

## Core brand idea

The strongest concise framing is:

**We do it all. AllDone.**

Expanded:

**We source it, pack it, bring it, collect it, and reuse it. You just use it.**

## Concept replacement rule

For materials derived from the earlier discussion:

* replace **PrecisionDelivery** with **AllDone**
* treat **glass packaging** as the default planned container standard
* treat **closed-loop reuse and recirculation** as the core system concept

## Short positioning statement

**AllDone is a closed-loop grocery and home provisioning system built around reusable glass packaging, collection, washing, and recirculation so the user does not have to shop, carry, sort, or throw packaging away.**

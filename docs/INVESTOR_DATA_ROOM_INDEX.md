# Investor Data Room Index

Status: active

Purpose
-------
This is the target structure for an AllDone investor data room.

It is meant to prevent ad hoc diligence and missing-document chaos.

## 00 Read Me First

- company purpose sentence
- short business summary
- last updated date
- primary contact
- document index

## Diligence rule

The point of the data room is to make AllDone easier to understand, not to spray every draft note at investors.

Canonical docs belong here.
Raw archives do not.

Local private folder note:

- the repo may use a gitignored `data_room_private/` folder for non-public diligence material

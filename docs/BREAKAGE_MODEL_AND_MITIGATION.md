# Breakage Model And Mitigation

Status: approved for pilot planning

Purpose
-------
This document defines the first breakage model and mitigation plan for the AllDone pilot.

It closes `P1.5`.

## Breakage zones

Breakage risk exists in four main zones:

- depot handling
- route transport
- household use
- returns

## Working assumption

Pilot one should be designed to keep combined breakage and non-return below `1%` of container turns.

This is a performance target, not a claim that the current system already meets it.

## Main failure modes

### Depot handling

- impact at sorting bench
- collisions in wash and drying zones
- rim damage during stacking or reassembly

### Route transport

- crate vibration
- container-to-container impact
- tipping or poor separation in vehicle or tote

### Household use

- sink-edge impact
- fridge-shelf collisions
- drop during hand carry

### Returns

- mixed loose returns
- closure mismatch
- dirty containers packed carelessly

## Mitigation measures

- use rack or crate systems rather than loose transport
- separate containers to reduce glass-to-glass collision
- standardize handling instructions for depot staff
- reject visibly damaged units immediately
- prefer stable stack geometry and flat top surfaces
- avoid pilot-only novelty shapes that are hard to handle

## Retirement thresholds

Retire immediately:

- chipped rim
- crack anywhere on glass body
- structural base damage

Do not return damaged units to circulation pending "one more use."

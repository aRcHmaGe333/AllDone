# Investor Pack

Purpose
-------
This folder is the public investor-pack structure for AllDone.

It is not a private diligence room.

The role of this folder is to make the fundraising surface concrete, legible, and easy to turn into outbound materials.

## Contents

- [ONE_PAGE_SUMMARY.md](ONE_PAGE_SUMMARY.md)
- [DECK_OUTLINE.md](DECK_OUTLINE.md)
- [USE_OF_FUNDS_18_MONTH.md](USE_OF_FUNDS_18_MONTH.md)
- [FAQ.md](FAQ.md)

## Relationship to the private data room

This public folder contains:

- narrative structure
- pitch framing
- non-sensitive fundraising assets

The private data room contains:

- sensitive diligence material
- counterpart names
- private terms
- working financial files
- outreach and negotiation materials
